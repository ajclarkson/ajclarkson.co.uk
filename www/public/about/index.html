About Me

About This Site

Everything you see before you has been handcoded with the help of Ruby, HTML5, CSS3, and Javascript, all pulled together by Jekyll. It gives me flexibility design every single page but still give you, the reader, a full featured ‘dynamic’ site.

The type is set in Freight Text Pro, with Franklin Gothic for the headings and meta data. Code examples are set in Source Code Pro, all fonts are served by Typekit.

A lot of the icons you can see on the site are provided by the excellent people at flat icon.
