ARMonkeyKit

JAVA Development, Augmented Reality, Graphics Programming, Framework Development

ARMonkeyKit is a rapid prototyping framework for Augmented Reality applications based on the NyARToolkit and jMonkeyEngine 2 libraries for JAVA. The framework provides a quick and easy way of creating fiducial marker based AR applications.

  • Build Stable Applications, Quickly

    The aim behind ARMonkeyKit was take away the heavy lifting of writing fiducial marker based AR systems, allowing the developer to concentrate on the content.

    Unfortunately ARMonkeyKit has not been updated to the latest version of jMonkeyEngine due to time constraints. It does however provide a clear example of how to decouple environment and feature tracking for AR, from the content aspects of the system.
