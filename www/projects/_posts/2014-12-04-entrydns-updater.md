<p>EntryDNS Updater is a Python utility which monitors the current public IP address and on detecting a change, updates a specified domain (or domains) via the EntryDNS API.</p>
</section>

<ul class="project-image-list project-list-2">
<li>
	<h2>Custom Dynamic DNS</h2>
	<p>Running services on your home server is great for your LAN, and remote access is possible on a non static IP if you add in Dynamic DNS. That often requires you to use a subdomain of some hosting company though. By registering your domain with EntryDNS you can use this handy update program to change the IP mapping of any number of hosts, every time your IP changes.</p>
	<p>A setup tutorial can be found on my <a href="/blog/custom-domain-dynamic-dns/">blog</a>.</p>
