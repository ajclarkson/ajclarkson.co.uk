<section class="project-info">
<p>Frankly was designed with simple writing in mind. It's a lightweight CMS based on the Ruby Sinatra framework (see what I did there?), aimed at taking a collection of articles in a git repository, applying a layout template and generating site navigation for them.</p>
</section>

<ul class="project-image-list project-list-2">
<li>
	<h2>Fully Git Controlled</h2>
	<p>Write your content in either HTML or markdown using whatever program or environment you are comfortable with, and it's all stored in version control so you can roll back whenever you want. Running a simple rake task builds your site automatically, and you publish by committing to a git repository. Content publishing for developers made simple.</p>

	<p><strong>Development is on hold at the minute as I found and started using Jekyll for this site, but I do plan to return to it in future and continue to make it the ultimate minimal writing CMS..</strong></p>
