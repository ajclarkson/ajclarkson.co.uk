# ajclarkson.co.uk

Personal site. Writing on home automation and engineering leadership.

## Stack

- [Astro](https://astro.build) v5 — static site generator
- [Netlify](https://netlify.com) — hosting and image optimisation
- MDX — content format

## Dev

```sh
npm install
npm run dev      # localhost:4321
npm run build
npm run preview
```

## Content

Posts live in `src/writing/{slug}/index.mdx`. See `CLAUDE.md` for content structure and frontmatter requirements.
