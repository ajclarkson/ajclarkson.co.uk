    • Make Blogging Simpler With Jekyll

    Providing a static page version of your blog reduces the runtime overheads, and provides a few other benefits along the way.

    Posted On

    14 Oct 2013

    Tagged

    • Jekyll
    • Git

    Disclaimer: Blogging with Jekyll requires a small amount of technical know-how, mainly focused around git. It's a great option for developers who blog, and designers who code which you should.

    Wordpress is pretty much the ubiquitous platform when it comes to setting up a blog now, and with good reason. As a blogging CMS, even as a full blown CMS, it does a great job. It makes it easy for users to post articles, and offers some powerful templating options and plugins. That makes it great for the masses. But just because it works for them, does that mean it has to work for you?

GIT_REPO=$HOME/ajclarkson.co.uk.git
TMP_GIT_CLONE=$HOME/tmp/ajclarkson.co.uk.git
