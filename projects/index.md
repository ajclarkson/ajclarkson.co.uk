## EntryDNS Updater
[EntryDNS Updater on Github](http://github.com/ajclarkson/entrydns-updater)

_EntryDNS Updater_ is a Python utility which monitors the current public IP address and on detecting a change, updates a specified domain (or domains) via the EntryDNS API.

Blog: [Dynamic DNS with a Custom Domain Name, EntryDNS &amp; Python](/blog/custom-domain-dynamic-dns)

## PyMap
[PyMap on Github](http://github.com/ajclarkson/pymap)

_PyMap_ is a basic Python IMAP Email processing program. At the moment it checks your 'Pending' emails folder for outstanding mail and emails you a friendly reminder that some of them need your attention.

It is very early stages of development at the moment, and I expect to add more functionality as the application evolves.


## FranklyCMS
[FranklyCMS on Github](http://github.com/ajclarkson/frankly)

_Frankly_ was designed with simple writing in mind. It's a lightweight CMS based on the Ruby Sinatra framework (see what I did there?), aimed at taking a collection of articles in a git repository, applying a layout template and generating site navigation for them.

It's development is on hold at the minute as I found and started using Jekyll for this site, but I do plan to return to it in future and continue to make it the ultimate minimal writing CMS.

## ARMonkeyKit
[ARMonkeyKit on Google Code](http://code.google.com/p/armonkeykit)

_ARMonkeyKit_ is a rapid prototyping framework for Augmented Reality applications based on the NyARToolkit and jMonkeyEngine 2 libraries for JAVA. The framework provides a quick and easy way of creating fiducial marker based AR applications.
